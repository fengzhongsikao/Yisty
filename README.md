#### 介绍
学习易经的网页，有桌面端，这是一个用vue做的易经学习程序，可以看卦象与爻辞

#### 软件架构
vue3 pinia bootstrap5 vite pnpm  nodejs

#### 安装说明
1. 安装nodejs
2. 安装pnpm
```java
npm install -g pnpm
```
3. clone项目
4. 在项目目录下执行这两行命令
 ```java
 pnpm install
pnpm run dev
 ```

 在浏览器打开即可看到运行效果
