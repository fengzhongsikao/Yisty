#### 介绍
学习易经的网页，有桌面端，这是一个用vue做的易经学习程序，可以看卦象与爻辞

#### 软件架构
vue3+pinia+element-plus
vite构建工具

#### 安装说明
1 在项目目录下执行 
 ```java
 pnpm install  

 ```
 2 在项目目录下执行 
 ```java
 pnpm run dev  

 ```
 即可看到运行效果
